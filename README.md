# Проект: Контейнеризирано Web Приложение

### Структура
* **Web (Flask):** Python бекенд, който брои посещенията.
* **DB (Redis):** База данни в паметта за съхранение на брояча.

### Комуникация
Услугите комуникират чрез вътрешна Docker мрежа. Web услугата достъпва базата данни чрез името `db`, дефинирано в `compose.yml`.

### Стартиране
1. Инсталирайте Docker Desktop.
2. Изпълнете `docker compose up --build`.
3. Отворете `http://localhost:8000`.
